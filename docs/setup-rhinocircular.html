This post will talk about setting up RhinoCircular.

System requirements

RhinoCircular is developed on:

  • Windows 10
  • Rhino 7
  • Visual Studio 2019

RhinoCircular is fully supported for:

  • Windows 10
  • Rhino 7

RhinoCircular is partially supported for:

  • macOS Catalina and above
  • Rhino 6 (Note: macOS and Rhino 6 are not officially fully supported due to the developers cannot reproduce the glitch if something wrong is happening on these platforms.)
